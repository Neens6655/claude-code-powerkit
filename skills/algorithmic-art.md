# /algorithmic-art — Generative Art with p5.js

Code that makes art. Math that becomes visual.

## Usage

```
/algorithmic-art [description]
```

Examples:
- `/algorithmic-art flowing particles that avoid the cursor`
- `/algorithmic-art concentric circles that pulse in sine waves`
- `/algorithmic-art a maze generated by recursive division`
- `/algorithmic-art noise-based terrain with color gradient`
- `/algorithmic-art a clock made of rotating geometric shapes`

## What Gets Produced

A self-contained `sketch.html` file using p5.js CDN. Opens in any browser, interactive if applicable.

## Core Patterns

### Noise-Based Organic Motion
```javascript
// Smooth, organic movement using Perlin noise
function draw() {
  const t = frameCount * 0.005
  for (let i = 0; i < particles.length; i++) {
    const angle = noise(particles[i].x * 0.003, particles[i].y * 0.003, t) * TWO_PI * 4
    particles[i].x += cos(angle) * 1.5
    particles[i].y += sin(angle) * 1.5
  }
}
```

### Recursive Geometry
```javascript
function branch(x, y, len, angle) {
  if (len < 4) return
  const x2 = x + cos(angle) * len
  const y2 = y + sin(angle) * len
  line(x, y, x2, y2)
  branch(x2, y2, len * 0.67, angle - PI/6)
  branch(x2, y2, len * 0.67, angle + PI/6)
}
```

### Grid + Noise Distortion
```javascript
for (let x = 0; x < width; x += cellSize) {
  for (let y = 0; y < height; y += cellSize) {
    const n = noise(x * 0.01, y * 0.01, frameCount * 0.003)
    const angle = n * TWO_PI * 2
    const len = cellSize * n
    line(x, y, x + cos(angle) * len, y + sin(angle) * len)
  }
}
```

## Tips

- `noiseDetail(4, 0.5)` — more detail in the noise (more complex patterns)
- `frameCount * 0.002` — vary this multiplier to change animation speed
- Save frames for exporting: `saveCanvas('frame-' + frameCount, 'png')` on keypress
- `randomSeed(42)` at the top of `setup()` makes sketches reproducible
- p5.js runs at 60fps by default — use `frameRate(30)` for complex sketches
