## The 7 Principles (Applied)

### 1. Loss Aversion
People fear losing more than they value gaining. Frame benefits as preventing loss.

```
❌ "Get more done with our app"
✅ "Stop losing 2 hours a day to disorganized tasks"
```

### 2. Social Proof
People copy what others do, especially in uncertain situations.

```
❌ "Trusted by businesses"
✅ "Trusted by 12,400 teams at Shopify, Notion, and Linear"
    (specificity + recognizable names = much stronger)
```

### 3. Anchoring
The first number seen becomes the reference point for all others.

```
// Annual plan looks cheap AFTER seeing monthly plan
Monthly: $49/month
Annual: $29/month (save 40%)
```

### 4. Reciprocity
Give something of value first. People feel obligated to return favors.

```
// Free tier, free guide, free audit, free trial
// All generate reciprocity before asking for anything
```

### 5. Scarcity
Limited availability increases perceived value.

```
❌ "Sign up today"
✅ "Join the early access list — 50 spots remaining"
```

### 6. Authority
Expertise signals and credentials reduce skepticism.

```
"Built by engineers from [FAANG], [YC company], [industry leader]"
"Featured in [publication]"
"[N] years of industry data behind every recommendation"
```

### 7. Specificity
Specific numbers are more credible than vague claims.

```
❌ "Saves you a lot of time"
✅ "Saves 4.3 hours per week (based on user survey, n=340)"
```

## Landing Page Audit

For any landing page, check:

### Above the Fold
- [ ] Headline answers: "What is this, who is it for, why should I care?"
- [ ] Subheadline addresses the primary objection
- [ ] CTA is visible without scrolling
- [ ] Social proof visible (logos, user count, rating)
- [ ] No jargon the target user wouldn't use

### Trust Signals
- [ ] Real company name and contact info
- [ ] Customer logos or testimonials
- [ ] Security badges (if payments involved)
- [ ] "No credit card required" near free trial CTA
- [ ] Money-back guarantee if applicable

### Conversion Killers
- [ ] Too many CTAs (pick one primary action)
- [ ] Long form above the fold
- [ ] Generic stock photos
- [ ] Vague benefits ("powerful", "flexible", "easy to use")
- [ ] No urgency or reason to act now

## Copy Patterns

### Hero Headline Formula
```
[Result] for [Audience] without [Obstacle]
→ "Launch marketing campaigns in minutes without a design team"
```

### Feature → Benefit → Emotion
```
Feature: "Automatic backups every 6 hours"
Benefit: "Your work is always safe"
Emotion: "Never lose a night's work again"
Always write the emotional layer — that's what converts.
```

### Testimonial Structure
```
"[Specific result] in [timeframe] — [social proof detail]"
→ "Cut our onboarding time from 3 days to 4 hours. We now close deals 2x faster."
   — [Name], [Title] at [Company]
```

## Tips

- The most common conversion killer: the page is about the product, not the user's problem
- Test the "so what?" test on every sentence: "Faster performance" → "So what?" → "Pages load in under 1 second" → "So what?" → "Visitors don't leave before seeing your offer"
- CTA copy: "Get started" is weak. "Start free trial", "Get my report", "Join 12,000 teams" are specific and action-oriented
