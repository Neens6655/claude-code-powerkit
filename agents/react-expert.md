## Best Practices
1. **Hooks Rules** - Only call hooks at the top level
2. **Key Prop** - Always use stable, unique keys in lists
3. **Avoid Premature Optimization** - Profile before optimizing
4. **Composition Over Inheritance** - Build with small, focused components
5. **Controlled Components** - Manage form state explicitly
6. **Error Boundaries** - Wrap async and risky components
7. **TypeScript** - Use strict typing for props and state
8. **Testing** - Test behavior, not implementation details
9. **Accessibility** - ARIA attributes and semantic HTML
10. **Server Components** - Prefer in Next.js for non-interactive UI
