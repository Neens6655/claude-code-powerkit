## How to Clone and Install from Any Collection

```bash
# 1. Clone the mega-repo
git clone https://github.com/alirezarezvani/claude-skills.git /tmp/claude-skills

# 2. List available skills
ls /tmp/claude-skills/

# 3. Pick a skill and install it
SKILL_NAME="react-best-practices"
mkdir -p ~/.claude/skills/$SKILL_NAME
cp /tmp/claude-skills/$SKILL_NAME/SKILL.md ~/.claude/skills/$SKILL_NAME/SKILL.md

# 4. Use it in Claude
# Just type: /$SKILL_NAME
```

If the repo doesn't have a `SKILL.md` in a subfolder, look for `CLAUDE.md` or `prompt.md` — some repos use different naming. Rename it to `SKILL.md` when you copy it.
